Мод для ослеживания метрик игрока и адаптации сложности под его уровень игры через адаптивного AI компаньона.
Для запуска:

  1)Скачать файл
  
  2)Вставить в корневую папку с модами \My Games\Terraria\tModLoader\ModSources\
  
  3)Включить мод внутри игры, используя специальную вкладку "Управление модами".
